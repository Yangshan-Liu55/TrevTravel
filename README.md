# TrevTravel – iOS Travel Diary App
Development period: 2018.05.02 – 2019.02.13

TrevTravel is an iOS application for creating and managing personal travel journals.
Built with Swift and powered by Firebase Firestore, Authentication, and Storage, the app focuses on clean UX, intuitive navigation, and reliable data handling for storing memories on the go.

## Features
### Travel Diary Management
- Create, edit, and delete travel journals
- List and organize all travel logs in a clean UI
- Store images in Firestore
### Comments on Travel Journals
- Add comments to individual travel journal entries
- Store comments in Firebase
- Real-time updates when new comments are added
- Simple UI for viewing and posting comments
### User Authentication
- Secure login and registration via Firebase Authentication
### Cloud-Based Storage
- Diary entries stored in Firebase
- Photos stored in Firebase Storage
